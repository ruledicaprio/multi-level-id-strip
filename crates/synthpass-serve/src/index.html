📄 SynthPass

Air-gapped document pipeline: pure-Rust OCR → Markdown → ICAO 9303 MRZ validation → local Qwen 2.5 GGUF fallback → structured JSON. Nothing leaves this machine.

🔒 No data is persistent on any server: uploads are processed in memory on this host and every working file is deleted the moment the response is sent. Images are downscaled in your browser before upload.

📷 Image formats only (JPEG, PNG, WebP, TIFF, BMP, GIF) — PDF and HEIC/HEIF are not supported.
